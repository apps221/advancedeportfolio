  • NFT Collection Virtual Internship

    HTML, CSS, JavaScript, React

    This was a website I worked on as part of my virtual internship at Frontend Simplified. I dynamically implemented different NFTs and their related details, including a timer countdown, onto the website from an API. It also has an Explore page where you can filter through NFTs and an item details and author page where you can read details about each NFT and the associated author.

  • ECommerce Book Library

    HTML, CSS, JavaScript, React

    This was an ecommerce platform for a library of books. Data was fetched from an internal API. It has a books page where you can filter through the books and a cart feature where you can add and remove the book from the cart.

  • YouTube Clone

    HTML, CSS, JavaScript, React

    This was a clone of YouTube I created using Rapid API to dynamically fetch data. I added a search bar and different side categories that a user can click on to access different categorized videos.

  • Netflix Clone

    HTML, CSS, JavaScript, Vite

    This was a Netflix clone I created using the TMDB API. It has user login, signup, and logout functionality. It also has a player page where users can watch the trailer for the selected movie based on data from the API.

  • Fitness Exercises App

    HTML, CSS, JavaScript, React

    This was a fitness application that loaded exercises from an exercise database on Rapid API to dynamically fetch the data. It has a horizontal scrollbar and search functionality. It also has separate exercise pages for each selected exercise dynamically.
